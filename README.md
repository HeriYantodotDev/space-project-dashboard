# Set Up Environment Variable

Under the server directory create a file `.env` There are several environment variable  :

PORT
MONGO_URL
ELON_PAST
CLIENT_ID
CLIENT_SECRET
COOKIE_KEY_1
COOKIE_KEY_2

Please set it up properly using `dotenv` package
